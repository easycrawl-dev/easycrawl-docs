# Home
Easycrawl is an AI-friendly web scraping tool that provides powerful capabilities to bypass Cloudflare and Google reCAPTCHA, allowing you to easily scrape content from protected websites.  
It's data structure is almost fully compatible with [Crawl4ai](https://github.com/unclecode/crawl4ai), enabling seamless migration of existing projects to Easycrawl.

> We recommend reading the [Crawl4ai Quickstart](https://docs.crawl4ai.com/quickstart/) to better understand the scraping process.
